Adding Elements(put)
    map.put(1, "Apple");
    map.put(2, "Banana");
    map.put(3, "Orange");

âš ï¸ If you add a value with an existing key, the old value is replaced:

-----------------------------------------------------------

Accessing Elements (get)

   String fruit = map.get(1); // Apple

If key does not exist:
    map.get(10); // returns null
-----------------------------------------------------------

Removing Elements
   map.remove(3); // removes key 3 and its value

-----------------------------------------------------------

Checking Keys and Values
    map.containsKey(1);     // true
    map.containsValue("Apple"); // true
    map.isEmpty();          // false
    map.size();             // number of entries

------------------------------------------------------------

Iterating Through a Map
a) Using entrySet() (Most common)

for (Map.Entry<Integer, String> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}

b) Using keySet()
for (Integer key : map.keySet()) {
    System.out.println(key + " = " + map.get(key));
}

c) Using values()
for (String value : map.values()) {
    System.out.println(value);
}
